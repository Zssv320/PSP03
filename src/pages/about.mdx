 ### 进入 https://www.ppsspp.org/
#
#

#

 - 4.点击应用程序
#

 ### 进入“游戏资源”板块 
#
 - 1.选择您喜欢的游戏并记住编号
#
 - 2.进入网盘🔗
#
 - 3.根据游戏编号找到下载链接
#
 - 4.下载完成后打开压缩包
#
  - 5.将解压密码“oldmanemu.net”复制进框
#
  - 6.创建一个空白的文件夹来保存游戏
#
  - 7.将解压好的内容拖进文件夹
#

#
 - 在载入新游戏中依次点击即可
